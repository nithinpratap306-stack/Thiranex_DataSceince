\# 📊 Data Cleaning \& Visualization Project



\## 📌 Overview

This project focuses on cleaning a raw dataset and extracting meaningful insights through data visualization. It demonstrates key data science skills such as preprocessing, handling missing values, and exploratory data analysis.

\## ⚙️ Technologies Used



\- Python  

\- Pandas  

\- Matplotlib  

\- Seaborn  



\---



\## 🧹 Data Cleaning



\- Missing values handled using median imputation  

\- Duplicate records removed  

\- Outliers treated using IQR method  

\- Dataset structured for analysis  



\---



\## 📊 Visualizations



\- Income Distribution  

\- Age vs Income  

\- Spending Score vs Income  

\- Credit Score vs Account Balance  

\- Correlation Heatmap  



\---



\## 🔍 Key Insights



\- Income distribution is right-skewed due to high-value customers  

\- Higher income is associated with higher spending behavior  

\- Credit score shows a strong relationship with account balance  

\- Transaction count increases with financial capacity
